# 소포수령증 자동화 시스템 — 설치 및 사용 방법

## 📁 폴더 구조
```
소포수령증_자동화/
├── input/               ← 소포수령증 PDF를 여기에 넣으세요
│   └── processed/       ← 처리 완료된 PDF가 자동으로 이동됨
├── output/              ← 결과 엑셀(매출집계_YYYYMM.xlsx) 저장
├── logs/                ← 실행 로그
├── modules/             ← 내부 모듈 (수정 불필요)
├── config.yaml          ← ★ 설정 파일 (큐텐 수동입력, 환율 등)
├── process.py           ← 처리 스크립트
├── watcher.py           ← 폴더 감시 자동화 스크립트
└── requirements.txt     ← 필요 패키지 목록
```

---

## 🔧 설치 (최초 1회)

### 1. Python 설치 확인
```bash
python --version   # Python 3.8 이상 필요
```

### 2. 필요 패키지 설치
```bash
pip install -r requirements.txt
```

---

## 🚀 사용 방법

### 방법 1: 폴더 감시 (자동) — 권장
```bash
python watcher.py
```
- 실행 후 `input/` 폴더에 PDF를 넣으면 **자동으로 처리**됩니다
- 결과 엑셀이 `output/` 폴더에 저장됩니다
- 종료: `Ctrl + C`

### 방법 2: 즉시 처리 (수동)
```bash
python process.py
```
- `input/` 폴더의 PDF를 지금 바로 처리합니다

### 방법 3: 연월 지정하여 처리
```bash
python process.py --year 2025 --month 12
```

## 📋 소포수령증 파일 넣는 방법

`input/` 폴더에 아래 파일들을 넣으세요:

| 파일 | 플랫폼 |
|------|--------|
| `유엠(UM)_MY_20251201-20251231.pdf` | 쇼피 말레이시아 |
| `유엠(UM)_PH_20251201-20251231.pdf` | 쇼피 필리핀 |
| `유엠(UM)_SG_20251201-20251231.pdf` | 쇼피 싱가폴 |
| `유엠(UM)_TH_20251201-20251231.pdf` | 쇼피 태국 |
| `유엠(UM)_TW_20251201-20251231.pdf` | 쇼피 대만 |
| `유엠(UM)_VN_20251201-20251231.pdf` | 쇼피 베트남 |
| `라자다_해외소포수령증_*.pdf` | 라자다 |
| `큐텐재팬_*.pdf` | 큐텐재팬 (※ 수동입력 필요) |

## ⚠️ 큐텐재팬 수동 입력

큐텐재팬 소포수령증은 이미지 기반 PDF라서 자동 추출이 불가합니다.
PDF를 직접 보고 `config.yaml`에 입력하세요:

```yaml
qoo10:
  jpy_amount: 3802685   # JPY 합계금액
  qty: 386              # 발송 건수
  tracking_no: "K2512244647017 외"
```

---

## 💱 환율 관련

- 환율은 **서울외국환중개(SMBS)** 에서 자동 수집합니다
  - URL: http://www.smbs.biz/ExRate/StdExRate.jsp
- **자동 수집 실패 시**: `config.yaml`의 `manual_rates`에 평균환율을 입력하면 해당 값으로 계산됩니다

```yaml
manual_rates:
  use_manual: false   # true로 바꾸면 항상 수동환율 사용
  MYR: 358.43
  PHP: 25.06
  ...
```

## 📊 결과물 (output/ 폴더)

생성되는 엑셀 파일 `매출집계_YYYYMM.xlsx` 에는 다음 시트가 포함됩니다:

| 시트 | 내용 |
|------|------|
| `총집계` | 전체 매출 집계 |
| `MYR`, `PHP`, `SGD`, `THB`, `TWD`, `VND`, `JPY` | 수출신고 프로그램 업로드용 템플릿 |
| `큐텐(소포수령증)` | 큐텐 소포수령증 내용 |
| `쇼피(MYR)` ~ `쇼피(VND)` | 쇼피 국가별 + 환율 적용 |
| `라자다(소포수령증)` | 라자다 소포수령증 |
| `라자다(MYR)` ~ `라자다(VND)` | 라자다 국가별 |
| `환율(JPY)` ~ `환율(VND)` | SMBS 환율 데이터 |
